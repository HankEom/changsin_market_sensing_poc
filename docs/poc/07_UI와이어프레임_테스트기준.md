## 4. PoC 테스트 체크리스트

### 기능 테스트

| # | 테스트 항목 | 기대 결과 | Pass |
|---|-----------|----------|------|
| T-01 | [수집 실행] 클릭 | 시드 데이터 30건 로드, 성공 토스트 | ☐ |
| T-02 | 대시보드 통계 표시 | 수집/키워드/매칭 카운트 정상 표시 | ☐ |
| T-03 | Top 키워드 바 차트 | CSS 프로그레스 바 + 변화율 Badge 정상 표시 | ☐ |
| T-04 | 키워드 테이블 정렬 | 트렌드 지수 내림차순 정렬 | ☐ |
| T-05 | [매칭 실행] 클릭 | 키워드별 매칭 용기 3건씩 표시 | ☐ |
| T-06 | 매칭 카드 적합도 | 0~100점 범위 내 표시 | ☐ |
| T-07 | [제안서 생성] 클릭 | 로딩 스피너 후 완성 제안서 Markdown 표시 | ☐ |
| T-08 | 제안서 복사 | 클립보드 복사 정상 동작 | ☐ |

### 시연 리허설 체크리스트

| # | 점검 항목 | 확인 |
|---|----------|------|
| R-01 | Vercel 배포 URL 접속 정상 | ☐ |
| R-02 | 수집→분석→매칭→제안서 전체 플로우 10분 이내 시연 가능 | ☐ |
| R-03 | LLM 응답 속도 체감 30초 이내 | ☐ |
| R-04 | 카드·테이블·바 차트 시각적 완성도 프로페셔널 수준 | ☐ |
| R-05 | 에러 발생 시 사용자 친화적 메시지 표시 | ☐ |

### 성공 기준 (Go/No-Go)

| 기준 | 충족 조건 |
|------|----------|
| **Go** | T-01~T-07 전체 Pass + R-01~R-04 확인 |
| **Conditional Go** | T-01~T-05 Pass + T-07 부분 동작 |
| **No-Go** | T-01~T-03 중 Fail |
