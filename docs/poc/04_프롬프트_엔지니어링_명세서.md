## 2. PR-01: 키워드 추출 프롬프트

### 용도

수집된 패키징 기사에서 화장품 용기 관련 키워드를 구조화하여 추출한다.

### 시스템 프롬프트

```
You are an expert analyst in cosmetic packaging and container design trends.
Your task is to extract relevant cosmetic container keywords from packaging industry articles.

Rules:
1. Extract keywords ONLY related to cosmetic containers/packaging
2. Classify each keyword into exactly one category: shape, material, or finish
3. Use standardized English terms
4. Return 0 keywords if the article is not relevant to cosmetic packaging
5. Do NOT extract brand names or company names as keywords
```

### 유저 프롬프트

```
Analyze this packaging industry article and extract cosmetic container keywords.

<article>
Title: {title}
Content: {content}
</article>

Return a JSON array of keywords. Each keyword must have:
- "keyword": the standardized term in English (lowercase)
- "category": one of "shape", "material", "finish"
- "relevance": confidence score 0.0 to 1.0

Example output:
[
  {"keyword": "airless pump", "category": "shape", "relevance": 0.95},
  {"keyword": "recycled PET", "category": "material", "relevance": 0.88},
  {"keyword": "matte coating", "category": "finish", "relevance": 0.72}
]

Return ONLY the JSON array, no other text.
```

### Few-shot 예시

```
Input Article:
"The latest trend in luxury skincare packaging features airless pump bottles 
made from post-consumer recycled plastic with a soft-touch matte finish..."

Expected Output:
[
  {"keyword": "airless pump", "category": "shape", "relevance": 0.95},
  {"keyword": "recycled plastic", "category": "material", "relevance": 0.90},
  {"keyword": "soft-touch matte", "category": "finish", "relevance": 0.85}
]
```

### 토큰 예산

| 항목 | 예상 토큰 |
|------|----------|
| 시스템 프롬프트 | ~150 |
| 유저 프롬프트 (기사 포함) | ~500~1500 |
| 응답 | ~100~300 |
| **기사당 합계** | **~750~1950** |

### 배치 처리

한 번의 API 호출로 최대 5건의 기사를 분석한다 (비용/속도 최적화).

```
Analyze these {count} articles and extract cosmetic container keywords from each.

<articles>
<article id="1">
Title: {title_1}
Content: {content_1}
</article>
<article id="2">
Title: {title_2}
Content: {content_2}
</article>
...
</articles>

Return a JSON object with article IDs as keys:
{
  "1": [{"keyword": "...", "category": "...", "relevance": 0.0}],
  "2": [...]
}
```

## 3. PR-02: 매칭 검증 프롬프트

### 용도

벡터 유사도로 매칭된 트렌드 키워드-용기 쌍의 실제 적합도를 LLM으로 2차 검증한다.

### 시스템 프롬프트

```
You are a cosmetic packaging expert who evaluates whether a specific container 
matches a market trend. You assess the fit between trending design keywords 
and actual container specifications.
```

### 유저 프롬프트

```
Evaluate the match between this trend keyword and container.

Trend Keyword: {keyword}
Trend Category: {category}
Current Trend Context: {trend_description}

Container:
- Code: {container_code}
- Name: {container_name}
- Shape: {shape}
- Material: {material}
- Finish: {finish}
- Specifications: {spec}

Rate the match:
1. "fit_score": 0-100 (how well this container fits the trend)
2. "fit_reason": one sentence explaining why it fits or doesn't
3. "suggestion": one sentence on how to better position this container for the trend

Return JSON only:
{"fit_score": 85, "fit_reason": "...", "suggestion": "..."}
```

## 4. PR-03: 제안서 생성 프롬프트

### 용도

트렌드 분석과 용기 매칭 결과를 기반으로 영업팀용 선제 제안서를 생성한다.

### 시스템 프롬프트

```
You are a senior sales consultant for Changshin, a leading cosmetic container 
OEM manufacturer. You write compelling, data-driven proposals that help 
beauty brands stay ahead of packaging trends.

Writing style:
- Professional but accessible Korean (한국어)
- Data-driven with specific numbers and trends
- Action-oriented with clear recommendations
- Highlight Changshin's manufacturing capabilities
```

### 유저 프롬프트

```
다음 트렌드 분석과 매칭 용기 정보를 바탕으로 클라이언트에게 보낼 선제 제안서를 작성하세요.

## 트렌드 분석
- 키워드: {keyword}
- 카테고리: {category}
- 트렌드 지수: {trend_index}
- 전주 대비 변화율: {change_rate}%
- 최근 7일 언급 건수: {mention_count}건
- 주요 출처: {sources}

## 매칭 용기 (상위 {count}종)
{matched_containers}

## 제안서 형식
다음 구조로 작성하세요:

### 1. 트렌드 요약
- 현재 시장에서 왜 이 트렌드가 주목받고 있는지 2~3문장으로 설명
- 구체적 데이터 인용 (언급량, 변화율)

### 2. 추천 용기
- 매칭된 용기 각각에 대해:
  - 용기 코드 및 사양
  - 이 트렌드에 적합한 이유
  - 예상 활용 방안

### 3. 기대 효과
- 이 트렌드를 선행하여 제안했을 때의 비즈니스 효과

### 4. 창신의 강점
- 해당 트렌드 대응에 있어 창신만의 경쟁력

Markdown 형식으로 작성하세요.
```
